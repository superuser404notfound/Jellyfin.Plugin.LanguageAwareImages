Language-Aware Images

Replaces the built-in TMDB image provider. Uses each library's metadata language by default and sorts images by vote_count within each language bucket, the same way TMDB's own UI ranks them.

Languages

Leave empty (recommended) to use each library's metadata language. Set a 2-letter code like de or fr to force that language for every library, ignoring per-library settings.
Used when no image in the preferred language exists.
Adds a third bucket between preferred and fallback. Useful for anime / foreign-cinema fans: a German user can see the Japanese poster for Princess Mononoke instead of the US export. Costs one extra API call per item.
Strict mode for posters: drops everything that isn't in the title's original language. For purists who want the original artwork no matter what their library language is. Implies "Also include the title's original language". Backdrops and logos are unaffected.

Allow textless (no language tag) images

Per image type, since logos and backdrops usually work fine textless but posters generally don't.

Sorting and quality

Sorts images within each language bucket by vote_count (most popular first), matching TMDB's own /images UI. When off, sorts by vote_average (highest rating first), which is Jellyfin's default behavior, but a single 5-star vote can outrank a 4.8-star image with 100 votes.
Drops images with fewer than this many TMDB votes. Default 0 keeps everything. A vote count of 0 usually means "not voted on yet" rather than "junk", and you don't want to fall back to English just because the German poster hasn't been rated. Bump to 1+ to aggressively trim unvalidated uploads.

Episode images

For shows with alternative orderings (Bluey, Star Trek, Doctor Who Classic etc.) where your library order doesn't match TMDB's. Smart mode: only intervenes when the TMDB title at (S, E) differs from your local title. In-sync shows still get the built-in provider's full image set. Costs one TMDB API call per season per show on first encounter, cached afterwards.

Image scaling

TMDB serves images at preset widths. Smaller saves bandwidth and disk; larger gives more resolution. Defaults match the official TMDB plugin (all original).

TMDB API

Leave empty to use Jellyfin's bundled key. Bring your own to avoid sharing rate limits.
